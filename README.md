# amazon-sentiment-analysis

Sentiment classifier built on 50,000 Amazon Fine Food Reviews.
## Models Used
- TF-IDF + Logistic Regression (baseline)
- DistilBERT fine-tuning (advanced)
## Results
- Baseline F1: ~85%
- DistilBERT F1: ~91%
## Dataset
Amazon Fine Food Reviews — Kaggle
## Tech Stack
Python, scikit-learn, HuggingFace Transformers, PyTorch, Pandas
